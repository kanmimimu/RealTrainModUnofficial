# RealTrainModUnofficial

このmodはRealTrainModの非公式1.21.1移植版です。
この度は配布許可をくださり、誠にありがとうございます。

## 📥 インストールガイド
このmodはNeoForge 1.21.1のみに対応しております。今後、対応バージョンが増える可能性があります。
このmodをダウンロードし、解凍せずにmodsフォルダへ導入してください。

## 📃問題の報告方法
バグ、クラッシュ、その他ご要望などは必ずRTMUのdiscordサーバーにてお願いします。
クラッシュが発生した場合は、必ずクラッシュログの送付をお願いいたします。クラッシュログがない場合、対応いたしかねます。
