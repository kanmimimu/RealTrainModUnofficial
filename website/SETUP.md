# rtmu.net への設置手順

このフォルダが新しい公式サイト一式。**中身をそのまま rtmu.net の公開ディレクトリへアップロードして
既存のファイルを置き換える** (文言は今のサイトと同じ、デザインとアニメーションだけ刷新)。

```
rtmu.net の公開ディレクトリ/
├── index.html        ← 新デザイン (文言は既存と同一)
├── styles.css        ← スタイル + アニメーション
├── script.js         ← スクロール演出 + バージョン自動表示
├── ban.txt           ← BAN リスト (RTMU の BAN 機能が読む)
└── assets/
    ├── rtmu.png      ← 既存ロゴをそのままコピー
    ├── hero.jpg      ← E257 の駅ホームのスクショ (トップ背景)
    └── interior.jpg  ← 車内のスクショ (概要セクション背景)
```

## アップロード前にやること

1. `assets/` に画像 3 枚を置く (`assets/画像をここに.txt` 参照)。
   - 画像が無くてもサイトは動く (背景がグラデーションになるだけ)。
2. ローカルで `index.html` をブラウザで開いて見た目を確認。

## 機能

- **バージョン表示の自動更新:** `v0.0.0` の表記は GitHub の最新リリースから自動で書き換わる
  (script.js)。リリースを作るたびにサイトを直す必要なし。取得に失敗したら書いてある表記のまま。
- **BAN 機能:** mod には `https://rtmu.net/ban.txt` が焼き込み済み。
  `ban.txt` を編集するだけで BAN の追加/削除 (1 行 1 ユーザー名、`#` はコメント、大文字小文字無視)。
  BAN されたユーザーは起動しても「あなたはRTMUをBANされています！」画面から先へ進めない。
  サーバーがダウンしていても安全側 (BAN しない)。
- **アップデート通知 (mod 側):** GitHub の最新リリースを見てタイトル画面右上にお知らせ。サイト側の作業なし。

## デザインの動き (参考)

- トップ: E257 のスクショがゆっくりズーム、E257 ダイヤ柄の粒子が浮遊、文字がせり上がり
- ヘッダー: スクロールでガラス化
- セクション: スクロールで順にフェードイン
- 区切り線: 走行灯が線路を流れる
- ボタン: ホバーで光が走る
- 導入方法: 路線図ふうの縦ライン + 駅パルス
