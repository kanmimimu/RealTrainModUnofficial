Guide

導入方法

1

1.21.1 の NeoForge を導入した起動構成を用意

Minecraft 1.21.1 で NeoForge が導入されている起動構成を、先に準備してください。

2

Mod の jar ファイルをダウンロード

GitHub の配布ページから、RealTrainMod Unofficial の jar ファイルをダウンロードしてください。

3

1.21.1 の NeoForge フォルダに jar ファイルを導入

ダウンロードした jar ファイルを、1.21.1 の NeoForge 環境の mods フォルダへ入れてください。
